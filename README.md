## 🛠️ Stack Tecnológico
Para este portfolio elegí herramientas que priorizan la performance y la experiencia de desarrollo (DX):

- **Framework:** [Astro](https://astro.build/) (Static Site Generator)
- **Estilos:** Tailwind CSS
- **Componentes:** Astro & React
- **Iconografía:** Lucide Astro
- **Deployment:** GitHub Actions (CI/CD)
- **Gestor de Paquetes:** pnpm v10
